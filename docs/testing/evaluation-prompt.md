# Prompt to Test the Project

Use this prompt in an evaluation run:

```text
Evaluate the AI News Intelligence Platform as a production-grade system.

Verify:
1. Multi-source ingestion coverage and freshness.
2. Deduplication quality for semantically similar stories.
3. Dashboard UX quality: performance, clarity, responsiveness.
4. Favorites workflow and database persistence.
5. Broadcast generation quality for Email, LinkedIn, WhatsApp.
6. Newsletter generation output quality and structure.
7. Vector search relevance for natural language queries.
8. Trend analytics correctness (topics, companies, papers, models).
9. Dockerized reproducibility and setup clarity.
10. API modularity, documentation, and error handling.

Provide scored feedback on:
- architecture quality
- AI integration depth
- scalability and reliability
- UI/UX quality
- codebase structure and maintainability
- deployment readiness

List: critical issues, major improvements, and launch readiness verdict.
```
